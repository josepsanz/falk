# falk
Tradueix al anglès:    Falk es concebut davant la necessitat d'estalviar el consum elèctric. Preten ser un planificador, controlador i gestor del consum elèctric d'un espai.  Falk was conceived due to the need to save electricity consumption. It aims to be a planner, controller, and manager of a space's electricity consumption
